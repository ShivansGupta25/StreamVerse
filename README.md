# 🎬 StreamVerse - Movie Explorer

StreamVerse is a modern and responsive movie discovery web application that allows users to explore trending movies, discover new films, search by title, and filter by genre, year, and ratings.
It uses the **TMDB (The Movie Database) API** to fetch real-time movie data and display it in a clean and interactive UI.

## ✨ Features

* 🔥 Trending Movies Carousel
* 🎬 Discover Movies Section
* 🔍 Search Movies by Title
* 🎭 Filter by Genre
* 📅 Filter by Year
* 📊 Sort by Rating / Popularity

* 🖼 Movie Posters with Fallback Image
